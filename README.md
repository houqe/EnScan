# 使用

提供了两种默认端口组，可以直接给到参数，或者 `-p` 指定其他端口

优先使用Icmp方式探活，可指定 `-ping` 参数

`-log` 默认为false，指定既输出日志到文件

不提供端口信息，默认只进行主机探活
